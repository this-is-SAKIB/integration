§ 00

What Integration Actually Means

Integration is the reverse of differentiation. If differentiation asks "what is the rate of change?", integration asks "given the rate of change, find the original function."

The symbol ∫ f(x) dx means: find a function whose derivative is f(x).

The Fundamental Relationship
If d/dx [F(x)] = f(x), then ∫ f(x) dx = F(x) + C
C is the constant of integration — it appears because the derivative of any constant is 0.
🔑 Always Remember
Every indefinite integral has a + C at the end. If you forget it, the answer is wrong, even if everything else is correct.

The dx at the end tells you which variable you are integrating with respect to. It also plays an essential role in the U-Substitution technique (Section 5).

§ 01

The Basic Rules (Always Apply These First)

Power Rule
∫ xⁿ dx = xⁿ⁺¹ / (n+1) + C
n ≠ −1
Constant Rule
∫ k dx = kx + C
Sum / Difference Rule
∫ [f±g] dx = ∫f dx ± ∫g dx
Constant Multiple Rule
∫ k·f(x) dx = k · ∫f(x) dx

The Power Rule — Step by Step

The pattern is simple: add 1 to the exponent, then divide by the new exponent.

Example 1.1 EASY
Find ∫ x⁵ dx
1
Identify the exponent: n = 5
2
Add 1 to the exponent: n + 1 = 6
x⁵ → x⁶
3
Divide by the new exponent (6):
x⁶ / 6
4
Add + C
x⁶/6 + C
Example 1.2 — Polynomial EASY
Find ∫ (3x⁴ − 5x² + 7) dx
1
Split using sum/difference rule — integrate each term separately.
2
Apply power rule to each term + constant rule to the 7:
∫3x⁴ dx → 3 · x⁵/5 = 3x⁵/5
∫5x² dx → 5 · x³/3 = 5x³/3
∫7 dx → 7x
3x⁵/5 − 5x³/3 + 7x + C
Example 1.3 — Negative & Fractional Exponents MEDIUM
Find ∫ (√x + 1/x²) dx
1
Rewrite using exponent notation — this is the most important first step!
√x = x^(1/2) and 1/x² = x^(-2)
2
Apply power rule to x^(1/2): add 1 to get 3/2, divide by 3/2
∫ x^(1/2) dx = x^(3/2) / (3/2) = (2/3)x^(3/2)
3
Apply power rule to x^(-2): add 1 to get -1, divide by -1
∫ x^(-2) dx = x^(-1) / (-1) = -x^(-1) = -1/x
(2/3)x^(3/2) − 1/x + C
💡 Trick: Rewrite Before Integrating
Always convert roots and fractions to exponent form before integrating:
∛x = x^(1/3)  ·  1/x³ = x^(-3)  ·  x√x = x · x^(1/2) = x^(3/2)
§ 02

Exponential Integrals

Core Formulas
∫ eˣ dx = eˣ + C

∫ e^(kx) dx = e^(kx) / k + C

∫ aˣ dx = aˣ / ln(a) + C
k is any non-zero constant. a is any positive base ≠ 1.

Why does ∫ eˣ dx = eˣ? Because eˣ is the only function that is its own derivative. So reversing that gives the same function.

Why does ∫ e^(kx) dx = e^(kx)/k? Because if you differentiate e^(kx), the chain rule brings down a factor of k. To undo that, we divide by k.

Example 2.1 EASY
Find ∫ e^(5x) dx
1
This is in the form e^(kx) with k = 5.
2
Use the formula: divide by k = 5.
e^(5x) / 5
e^(5x)/5 + C
Example 2.2 EASY
Find ∫ 3ˣ dx
1
This is in the form aˣ with a = 3.
2
Use the formula: divide by ln(a) = ln(3).
3ˣ / ln(3)
3ˣ / ln(3) + C
Example 2.3 — Mixed MEDIUM
Find ∫ (4eˣ − 2e^(3x) + 5ˣ) dx
1
Split and integrate each term:
2
∫4eˣ dx = 4eˣ
∫2e^(3x) dx = 2 · e^(3x)/3 = (2/3)e^(3x)
∫5ˣ dx = 5ˣ/ln(5)
4eˣ − (2/3)e^(3x) + 5ˣ/ln(5) + C
§ 03

Logarithmic Integrals

Core Formulas
∫ (1/x) dx = ln|x| + C

∫ (1/(ax+b)) dx = (1/a) · ln|ax+b| + C

∫ ln(x) dx = x·ln(x) − x + C
Always use absolute value |x| inside the ln. Why? Because ln only accepts positive numbers, but 1/x can appear for negative x too.

The formula ∫ (1/x) dx = ln|x| is the special case of the power rule that the power rule CANNOT handle (when n = −1, you'd be dividing by zero).

Example 3.1 EASY
Find ∫ 1/(3x+4) dx
1
This matches the form 1/(ax+b) with a=3, b=4.
2
Divide by a = 3:
(1/3) · ln|3x+4|
(1/3)·ln|3x+4| + C
Example 3.2 — Hidden 1/x MEDIUM
Find ∫ (x² + 3)/x dx
1
Split the fraction first — divide each term by x:
(x²/x) + (3/x) = x + 3/x
2
Now integrate each term:
∫x dx + ∫(3/x) dx = x²/2 + 3·ln|x|
x²/2 + 3·ln|x| + C
Example 3.3 — Numerator is derivative of denominator (KEY PATTERN) HARD
Find ∫ (2x)/(x²+5) dx
1
Notice: The derivative of the denominator (x²+5) is 2x, which is exactly the numerator. This is a clue that the answer is a ln!
2
Whenever you see ∫ f'(x)/f(x) dx, the answer is ln|f(x)| + C.
Here f(x) = x²+5 and f'(x) = 2x
ln|x²+5| + C
🔑 Golden Pattern for Fractions
If the numerator is the derivative of the denominator (or a multiple of it):
∫ f'(x) / f(x) dx = ln|f(x)| + C
This pattern appears constantly in trick questions. Always check if the top is the derivative of the bottom!

sin(4x)/4 + C

§ 05

U-Substitution — The Chain Rule in Reverse

🔑 The Big Idea
When differentiating, the chain rule says: derivative of f(g(x)) = f'(g(x)) · g'(x).

U-substitution reverses this. When you see a function composed with another function, and the derivative of the inner function is also somewhere in the integral, you can substitute.
The U-Substitution Recipe
Step 1: Let u = [inner function]
Step 2: Find du/dx → rearrange to get du = (...) dx
Step 3: Substitute u and du into the integral (x should disappear)
Step 4: Integrate in terms of u
Step 5: Replace u back with the original expression in x

How to spot when to use U-Sub

Look for a composite function — a function inside another function. Also check if the derivative of the inner function appears elsewhere in the integral (sometimes multiplied by a constant).

Example 5.1 — Classic U-Sub EASY
Find ∫ 2x · (x²+1)⁵ dx
1
Identify the inner function: the thing raised to the 5th power is (x²+1). Let u = x²+1.
u = x² + 1
2
Differentiate u with respect to x, then solve for dx:
du/dx = 2x → du = 2x dx
3
Substitute into the integral. Notice 2x dx appears in our integral — replace it with du:
∫ 2x · (x²+1)⁵ dx = ∫ u⁵ du
4
Integrate using the power rule:
∫ u⁵ du = u⁶/6
5
Replace u back:
u⁶/6 = (x²+1)⁶ / 6
(x²+1)⁶ / 6 + C
Example 5.2 — Adjusting for a Missing Constant MEDIUM
Find ∫ x · e^(x²) dx
1
Inner function of the exponent is x². Let u = x².
u = x²
2
du/dx = 2x → du = 2x dx
But we only have x dx in the integral, not 2x dx. So we adjust:
x dx = du/2
3
Substitute:
∫ x · e^(x²) dx = ∫ eᵘ · (du/2) = (1/2) ∫ eᵘ du
4
(1/2) · eᵘ
5
Back-substitute:
(1/2) · e^(x²)
(1/2)e^(x²) + C
Example 5.3 — U-Sub with Trig MEDIUM
Find ∫ sin(x) · cos⁴(x) dx
1
The inner function is cos(x) (inside the 4th power). Let u = cos(x).
u = cos(x)
2
du/dx = −sin(x) → du = −sin(x) dx → sin(x) dx = −du
3
Substitute (sin(x) dx becomes −du):
∫ sin(x) · cos⁴(x) dx = ∫ u⁴ · (−du) = −∫ u⁴ du
4
−u⁵/5
5
Back-substitute:
−cos⁵(x)/5
−cos⁵(x)/5 + C
Example 5.4 — U-Sub with Fractions (ln pattern) MEDIUM
Find ∫ (3x²)/(x³−7) dx
1
The denominator (x³−7) has derivative 3x² which is the numerator! Classic log pattern. Set u = x³−7.
u = x³ − 7
2
du = 3x² dx
3
∫ (3x²)/(x³−7) dx = ∫ (1/u) du = ln|u|
ln|x³ − 7| + C
⚠️ Common Mistake
When you multiply or divide to adjust the constant, you can only do this if the missing piece is a constant, not a variable. You cannot multiply by x inside and divide by x outside.
§ 06

Integration by Parts — The Product Rule in Reverse

🔑 The Big Idea
The product rule for differentiation says: d/dx[u·v] = u·v' + v·u'
Rearranging and integrating both sides gives us Integration by Parts. Use this when you have a product of two different types of functions.
Integration by Parts Formula
∫ u · dv = u·v − ∫ v · du
You choose what to call u and what to call dv. The right choice makes the problem easier. Use LIATE to choose.

LIATE — How to Choose u

Choose u from the category that comes first in this list. The other function becomes dv.

L
Logarithms
ln(x), log(x)
I
Inverse Trig
arctan, arcsin
A
Algebraic
x, x², x³
T
Trigonometric
sin(x), cos(x)
E
Exponentials
eˣ, 2ˣ
Example 6.1 — Classic: x·eˣ EASY
Find ∫ x·eˣ dx
1
We have x (Algebraic) and eˣ (Exponential). By LIATE, choose A before E, so:
u = x → du = dx
dv = eˣ dx → v = eˣ
2
Apply the formula ∫ u dv = uv − ∫ v du:
= x·eˣ − ∫ eˣ dx
3
Solve the remaining integral:
= x·eˣ − eˣ
xeˣ − eˣ + C = eˣ(x − 1) + C
Example 6.2 — With Logarithm MEDIUM
Find ∫ x·ln(x) dx
1
We have ln(x) (Logarithm) and x (Algebraic). By LIATE, L comes before A, so:
u = ln(x) → du = (1/x) dx
dv = x dx → v = x²/2
2
Apply the formula:
= ln(x)·(x²/2) − ∫ (x²/2)·(1/x) dx
3
Simplify the remaining integral: (x²/2)·(1/x) = x/2
= (x²/2)·ln(x) − ∫ (x/2) dx
= (x²/2)·ln(x) − x²/4
(x²/2)·ln(x) − x²/4 + C
Example 6.3 — Applying Twice (Tabular Method) MEDIUM
Find ∫ x²·eˣ dx
1
By LIATE: u = x², dv = eˣ dx. The tabular method is faster when you apply by parts multiple times. Build a table:

Differentiate the u-column, integrate the dv-column, alternate signs:
Signs: + Differentiate (u): x² | Integrate (dv): eˣ
Signs: − Differentiate: 2x | Integrate: eˣ
Signs: + Differentiate: 2 | Integrate: eˣ
Signs: − Differentiate: 0 | Integrate: eˣ (STOP)
2
Multiply diagonally (sign × left term × right term from next row) and add:
(+1)·x²·eˣ + (−1)·2x·eˣ + (+1)·2·eˣ
x²eˣ − 2xeˣ + 2eˣ + C = eˣ(x² − 2x + 2) + C
Example 6.4 — The Cycling Trick (eˣ·sin/cos) HARD
Find ∫ eˣ·sin(x) dx
1
Call the integral I. Apply by parts with u = sin(x), dv = eˣ dx:
u = sin(x) → du = cos(x) dx
dv = eˣ dx → v = eˣ
I = eˣ·sin(x) − ∫ eˣ·cos(x) dx
2
Apply by parts AGAIN to ∫ eˣ·cos(x) dx with u = cos(x), dv = eˣ dx:
u = cos(x) → du = −sin(x) dx
dv = eˣ dx → v = eˣ
∫ eˣ cos(x) dx = eˣ cos(x) − ∫ eˣ·(−sin(x)) dx = eˣcos(x) + ∫eˣsin(x) dx
3
Substitute back: I = eˣsin(x) − [eˣcos(x) + I]
We got I on both sides! Solve for I:
I = eˣsin(x) − eˣcos(x) − I
2I = eˣsin(x) − eˣcos(x)
I = eˣ(sin(x) − cos(x)) / 2
eˣ(sin(x) − cos(x)) / 2 + C
💡 When to Use Integration by Parts
Use it when the integral is a product of two different types of functions (polynomial × exponential, polynomial × trig, logarithm × anything). U-sub won't work because neither factor is the derivative of the other.
§ 07

Partial Fractions — For Rational Functions

When you have a fraction where the denominator factors into linear pieces, you can split it into simpler fractions that are easy to integrate.

The Setup
A/(ax+b) + B/(cx+d) → each part integrates as a ln
Degree of numerator must be less than denominator. If not, do polynomial long division first.
Example 7.1 — Two Distinct Linear Factors MEDIUM
Find ∫ 1/(x²−x−2) dx
1
Factor the denominator:
x²−x−2 = (x−2)(x+1)
2
Set up the partial fraction decomposition:
1/[(x−2)(x+1)] = A/(x−2) + B/(x+1)
3
Multiply both sides by (x−2)(x+1):
1 = A(x+1) + B(x−2)
4
Find A and B by plugging in smart values:
Let x = 2: 1 = A(3) + B(0) → A = 1/3
Let x = −1: 1 = A(0) + B(−3) → B = −1/3
5
Substitute back and integrate:
∫ [(1/3)/(x−2) + (−1/3)/(x+1)] dx
= (1/3)ln|x−2| − (1/3)ln|x+1|
(1/3)·ln|x−2| − (1/3)·ln|x+1| + C
Example 7.2 — Repeated Factor HARD
Find ∫ (x+1)/(x²(x−3)) dx
1
The denominator has a repeated factor x² (and x−3). When a factor is repeated, include EACH power:
(x+1)/(x²(x−3)) = A/x + B/x² + C/(x−3)
2
Multiply through by x²(x−3):
x+1 = Ax(x−3) + B(x−3) + Cx²
3
Let x=0: 1 = B(−3) → B = −1/3
Let x=3: 4 = C(9) → C = 4/9
Expand and match x² coefficient for A: 0 = A + C → A = −4/9
4
Integrate each term:
∫(−4/9)/x dx + ∫(−1/3)/x² dx + ∫(4/9)/(x−3) dx
= (−4/9)ln|x| + (1/3)(1/x) + (4/9)ln|x−3|
(−4/9)ln|x| + 1/(3x) + (4/9)ln|x−3| + C
§ 08

Completing the Square

When you have a quadratic in the denominator that doesn't factor, complete the square to transform it into a recognizable form.

Two Key Results After Completing the Square
∫ 1/(x² + a²) dx = (1/a)·arctan(x/a) + C

∫ 1/√(a²−x²) dx = arcsin(x/a) + C
These appear constantly in exam trick questions. Memorize them.
Example 8.1 — Complete the Square to Use arctan HARD
Find ∫ 1/(x²+4x+13) dx
1
The denominator doesn't factor nicely. Complete the square:
x²+4x+13 = (x²+4x+4) + 9 = (x+2)² + 9 = (x+2)² + 3²
2
Substitute u = x+2, so du = dx:
∫ 1/(u² + 3²) du
3
Use the formula with a = 3:
(1/3)·arctan(u/3)
4
Back-substitute u = x+2:
(1/3)·arctan((x+2)/3) + C
§ 09

Trig Identity Tricks — Powers of sin and cos

When integrating powers of sin or cos, use these identities to reduce the powers first.

The Two Essential Power-Reduction Identities
sin²(x) = (1 − cos(2x)) / 2

cos²(x) = (1 + cos(2x)) / 2

sin(x)·cos(x) = sin(2x) / 2
These turn a squared trig function into something you can integrate directly.
Example 9.1 — ∫ sin²(x) MEDIUM
Find ∫ sin²(x) dx
1
You cannot integrate sin²(x) directly. Use the identity:
sin²(x) = (1 − cos(2x)) / 2
2
∫ (1 − cos(2x))/2 dx = (1/2)∫ dx − (1/2)∫ cos(2x) dx
3
= x/2 − (1/2)·sin(2x)/2 = x/2 − sin(2x)/4
x/2 − sin(2x)/4 + C
Example 9.2 — Odd Power of sin MEDIUM
Find ∫ sin³(x) dx
1
For odd powers, peel off one factor and convert the rest using sin²(x) = 1 − cos²(x):
sin³(x) = sin²(x)·sin(x) = (1 − cos²(x))·sin(x)
2
∫ (1 − cos²(x))·sin(x) dx
Use U-sub: u = cos(x), du = −sin(x) dx:
−∫ (1 − u²) du = −u + u³/3
3
Back-substitute:
−cos(x) + cos³(x)/3 + C

Key Trig Identities to Remember

Pythagorean
sin²(x) + cos²(x) = 1
1 + tan²(x) = sec²(x)
Double Angle
sin(2x) = 2sin(x)cos(x)
cos(2x) = cos²(x)−sin²(x)
§ 10

Trick Questions — The Most Common Types

Trick Type 1: Disguised Power Rule (Rewriting the Fraction)

Example T.1 HARD
Find ∫ (x³ − 2x + 5) / √x dx
1
Rewrite √x = x^(1/2) and divide each term:
x³/x^(1/2) = x^(5/2)
2x/x^(1/2) = 2x^(1/2)
5/x^(1/2) = 5x^(−1/2)
2
Apply power rule to each:
x^(7/2)/(7/2) − 2x^(3/2)/(3/2) + 5x^(1/2)/(1/2)
= (2/7)x^(7/2) − (4/3)x^(3/2) + 10x^(1/2)
(2/7)x^(7/2) − (4/3)x^(3/2) + 10√x + C

Trick Type 2: Add and Subtract in Numerator

Example T.2 HARD
Find ∫ x/(x+1) dx
1
The degree of numerator = degree of denominator. Do a clever rewrite — add and subtract 1 in the numerator:
x/(x+1) = [(x+1) − 1]/(x+1) = 1 − 1/(x+1)
2
Now integrate:
∫ [1 − 1/(x+1)] dx = x − ln|x+1|
x − ln|x+1| + C

Trick Type 3: Spotting the Derivative of the Denominator

Example T.3 HARD
Find ∫ sin(x)·ecos(x) dx
1
The inner function of eᶜᵒˢ⁽ˣ⁾ is cos(x). Its derivative is −sin(x), and we have sin(x) in the integral!
2
Let u = cos(x) → du = −sin(x) dx → sin(x) dx = −du
∫ eᵘ · (−du) = −eᵘ
−e^cos(x) + C

Trick Type 4: Long Division First

Example T.4 HARD
Find ∫ (x³ + 1)/(x + 1) dx
1
The numerator has higher degree than the denominator, so divide first. Factor the numerator:
x³ + 1 = (x+1)(x² − x + 1)
2
Cancel (x+1):
(x³+1)/(x+1) = x² − x + 1
3
Now integrate normally:
∫ (x² − x + 1) dx = x³/3 − x²/2 + x
x³/3 − x²/2 + x + C

Trick Type 5: Multiplying by a Clever Form of 1

Example T.5 HARD
Find ∫ 1/(1 + eˣ) dx
1
Multiply numerator and denominator by e^(−x):
1/(1+eˣ) · (e^(−x)/e^(−x)) = e^(−x)/(e^(−x) + 1)
2
The numerator is e^(−x), which is almost the derivative of the denominator (e^(−x)+1). d/dx(e^(−x)+1) = −e^(−x). So we're off by a minus sign:
∫ e^(−x)/(e^(−x)+1) dx = −ln|e^(−x)+1|
−ln(e^(−x) + 1) + C

Trick Type 6: U-Sub Where u Replaces Everything

Example T.6 HARD
Find ∫ eˣ/(eˣ + 1) dx
1
The numerator eˣ is exactly the derivative of the denominator (eˣ+1).
2
Let u = eˣ+1 → du = eˣ dx:
∫ (1/u) du = ln|u|
ln(eˣ + 1) + C

Trick Type 7: Integration by Parts on ln(x) Alone

Example T.7 HARD
Find ∫ ln(x) dx
1
Rewrite as a product: ln(x) · 1. Then apply by parts with u = ln(x), dv = 1 dx:
u = ln(x) → du = (1/x) dx
dv = dx → v = x
2
Apply the formula:
= x·ln(x) − ∫ x · (1/x) dx = x·ln(x) − ∫ 1 dx
x·ln(x) − x + C

Trick Type 8: Definite Integral with Odd/Even Symmetry

🔑 Symmetry Shortcuts for Definite Integrals
If the limits are symmetric about 0, i.e., from −a to +a:

• If f(x) is odd [f(−x) = −f(x)]:   ∫₋ₐᵃ f(x) dx = 0 (the two halves cancel)
• If f(x) is even [f(−x) = f(x)]:   ∫₋ₐᵃ f(x) dx = 2∫₀ᵃ f(x) dx

Examples of odd functions: x, x³, sin(x), tan(x)
Examples of even functions: x², x⁴, cos(x), |x|
Example T.8 HARD
Find ∫₋₃³ (x⁵ + 3x³ − 2x) dx
1
Limits are −3 to 3, which is symmetric about 0.
2
Check: x⁵, x³, x are all odd functions. Sum of odd functions is odd.
3
Therefore the integral = 0. No calculation needed!
0

Trick Type 9: Recognizing eˣ + e^(−x) / eˣ − e^(−x) Patterns

Example T.9 HARD
Find ∫ (eˣ − e^(−x))/(eˣ + e^(−x)) dx
1
The numerator is the derivative of the denominator! d/dx(eˣ + e^(−x)) = eˣ − e^(−x). ✓
2
Direct application of f'(x)/f(x) = ln|f(x)|:
ln(eˣ + e^(−x)) + C

Decision Tree — Which Technique to Use?

Step-by-Step Decision
1. Can you integrate directly with a basic formula? → Do it.
2. Can you rewrite the expression (split fraction, use exponents)? → Rewrite, then integrate.
3. Is the numerator the derivative of the denominator? → Answer is ln|denominator|.
4. Is there a composite function where the inner function's derivative also appears? → U-Substitution.
5. Is it a product of two different function types (poly × trig, poly × exp, ln × anything)? → Integration by Parts (LIATE).
6. Is the denominator a factorable polynomial? → Partial Fractions.
7. Is the denominator a quadratic that won't factor? → Complete the Square.
8. Are you integrating sin²(x) or cos²(x)? → Use power-reduction identity.
9. Is it eˣsin(x) or eˣcos(x) type? → Integration by Parts twice, then solve for I.
💡 Final Advice
The secret to integration is pattern recognition. The more examples you do, the more patterns you see instantly. Every trick question is just a standard formula in disguise. When stuck: rewrite the expression, look for f'(x)/f(x), or try U-sub with the most complicated part as u.
